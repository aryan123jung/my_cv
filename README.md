# Aryan Jung Chhetri Portfolio & CV

A premium dark portfolio website built with Next.js, Tailwind CSS, Framer Motion, and Lucide icons.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. Open Vercel and choose **New Project**.
3. Import the repository.
4. Keep the defaults:
   - Framework: Next.js
   - Build command: `npm run build`
   - Output directory: `.next`
5. Deploy.

## Customization

Primary portfolio content lives in `data/profile.ts`. Replace the placeholder GitHub and LinkedIn URLs with Aryan's real profiles when available.
